The Antibiotic Resistance Microbiology Dataset Mass General Brigham (ARMD-MGB) is a de-identified 
resource derived from electronic health records (EHR) that facilitates research in antimicrobial 
resistance (AMR).[7] AMR is a well-known, severe and complex healthcare issue that endangers modern 
societies and may be able to be reduced in complexity through research utilizing Machine Learning (ML) 
algorithms. This data set was developed with this in mind and appropriately de-identified in order to 
make the AMR information more accessible to researchers and to protect patient privacy. We are building 
an integrated framework to enable researchers to interact at-a-glance with the trends and aggregate 
data presented within this dataset. To this end, we are using Bootstrap JS with a Python backend on 
top of a MySQL database. This visual-first approach reduces handling time from hours or days to minutes 
or hours, depending on the depth of conclusions necessary, and can enable researchers from disparate 
disciplines to participate without having to train in a technical background.
